Author: loannes Date: 一月 19, 2020  17:33:18

如何创建链接

  1. 把链接文本放在元素中
  2. 通过href属性指定链接的目标文件

浏览器做了什么

  1. 浏览器如果遇到了元素，会把该元素的内容显示为可单击的链接。
  2. 用户单击链接，浏览器通过href属性确定跳往哪个页面

小结

  • 想从一个页面跳转到另外一个页面，要使用元素
  • 元素中的href属性指定了链接的目标文件
  • 文字和图像都可用作链接的标签
  • 相对路径是相对于链接的源Web页面指向网站中其他文件的一个链接。
  • 使用”..”可以链接到源文件上一层文件夹中的文件。
