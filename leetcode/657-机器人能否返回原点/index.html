657. 机器人能否返回原点

2020-12-06

前往此题

解题思路

设定x,y变量代表坐标系
L,R,U,D分别对应x,y变量的加减。

如果指令是 UU，则令 y=y-1y=y−1
如果指令是 DD，则令 y=y+1y=y+1
如果指令是 LL，则令 x=x-1x=x−1
如果指令是 RR，则令 x=x+1x=x+1

最后再判断是否在原点即可。

代码

class Solution:
    def judgeCircle(self, moves: str) -> bool:
        x = 0
        y = 0
        for i in moves:
            if i == 'L': x -= 1
            if i == 'R': x += 1
            if i == 'U': y += 1
            if i == 'D': y -= 1
        if x == 0 and y == 0: return True
        return False
