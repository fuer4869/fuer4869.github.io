Author: loannes Date: 十一月 4, 2019  16:00:05 Category: HTML

引用元素q 和 blockquote

<q> 元素以及 <blockquote>元素都是用来表示引用。一般浏览器都是用引号来表示
如：

最终显示:
“abc”
同样 <blockquote>也是。

那么引用元素与直接在段落中加上引号有什么区别呢？

如果直接放在段落<p>中并且手动加上引号，会造成浏览器会默认为这一段是段落中的内容。所以即使你在后面修改了引号的样式，也不会起到作用。所以为了方便后面对样式的修改，必须严格按照规范，什么样的内容放在适合的元素中。

元素q与blockquote有什么区别？

比如在一个段落中，加上引用。

用<q>元素

<p>Well I made it 1200 miles already, and I passed through some interesting places on the way: <q>Passing cards,When you cant see,May get you, A glimpse,Of eternity.</q>``</p>

用<blockquote>元素

<p>Well I made it 1200 miles already, and I passed through some interesting places on the way: <blockquote>Passing cards,When you cant see,May get you, A glimpse,Of eternity.</blockquote>``</p>

会发现，使用blockquote元素的内容会在段落中形成新的段落，成为了独立的块。
所以在开发过程中需要看情况而定，使用这两种不同的引用元素。q元素会在文本中形成一个引用，而blockquote则会另外起一个段落，形成一个独立块。

所以可以这么说<q>元素是内联元素

而<blockquote>元素是块元素

所有内联元素可以存在于块元素中，而不会重新起一行，可谓是随波逐流比较随和。
而块元素比较特立独行，不会融于任何块元素及内联元素。就像<blockquote>一样会新起一样，形成新的块。

列表

如何使用列表

  1. 将每个列表项放入<li>元素中
  2. 用< ol>元素或<ul>元素将列表项包围起来。

unordered list = ul 无序列表

ordered list = ol 有序列表

list item = li 列表项

特殊字符

在Web页面中如何表示<和>符号？

&lt; 表示 <
&gt; 表示 >
&amp; 表示 &
