Head First HTML&CSS---Web语言简介

Author: loannes Date: 十一月 2, 2019  16:43:45 Category: HTML

要点

  • HTML和CSS是我们用来创建网页的语言，前者负责网页的结构，后者负责网页的样式，是两种不同的语言。
  • Web服务器存储并提供由HTML和CSS创建的网页。浏览器获取页面，并根据HTML和CSS显示网页的内容
  • HTML是超文本标记语言（HyperText Markup Language)的缩写，用来建立网页的结构。
  • CSS是层叠样式表，用来控制HTMl的表现。
  • 通过HTML，我们利用标记来标示内容提供结构。我们把匹配标记以及它们包围的内容称为元素。
  • 元素由3部分组成：一个开始标记、内容和一个结束标记。部分元素例外如<img>
  • 开始标记可以有属性。
  • 所有页面都要有一个<html>元素，其中要有一个<head>元素和一个元素
  • 网页的信息应该放在<head>元素里
  • 可以在<style>元素中写CSS规则，为HTML网页增加CSS。<sytle>元素总要放在<head>元素里。
  • <style>元素中的body属性用来指定HTML页面中<body>元素的样式
  • 可以使用CSS在HTML中指定元素的特性
