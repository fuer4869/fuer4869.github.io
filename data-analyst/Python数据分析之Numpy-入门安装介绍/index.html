Python数据分析之Numpy---入门安装介绍

2019-04-11

NumPy(Numerical Python)是高性能科学计算和数据分析的基础包，主要功能是对N维数组的操作和计算，其计算效率是Python原生包的N倍，在使用Python做数据分析过程中NumPy是我们必不可少的一个工具。

作为一个基础包，功能上远不及pandas来的那么高级（下面会出pandas系列的笔记），但是作为新手入门NumPy还是很值得我们学习一下，为接下来学习pandas打好基础。

准备工作

  • 安装Python环境
    如果没有pip环境请参考这篇文章 pip 的简单安装与基本使用

windows

下载NumPy安装包，注意安装包的Python版本和系统版本。例如numpy-1.16.2-cp27-cp27m-manylinux1_x86_64.whl中cp27指的是python2.7版本。

需先进入Python的根目录中然后执行以下命令

   >>> pip install D:\source\numpy-1.11.2-cp27-none-win_amd64.whl

最后的目录因人而异，但必须输入安装包的全路径。接下来就等命令行中安装成功的提示吧。
如果还不放心，为了验证安装是否成功，可在Python中进行如下操作

  >>> import numpy as np

如果系统没有提示任何异常说明安装成功了。

Mac OSX

因为Mac 本身自带python2.7环境，建议安装python3.4以上版本，因为它自带了pip安装包管理器，安装其他工具十分方便。
接下来输入以下命令即可

  >>> pip3 install numpy
